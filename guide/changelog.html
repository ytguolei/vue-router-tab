# Changelog

# 0.2.9 (2019-11-20)

# Bug Fixes

  • operate: adopt fuzzy matching when force-new opening tabs in case that subtle distinction in route path might prevent force reloading (150f112)

# 0.2.8 (2019-11-15)

# Bug Fixes

  • operate: fix errors with closing tabs in a way of close(path, to, match, force) under IE 11, which is due to the uncompatibility of destructuring assignment (acb7e7a)

# Code Refactoring

  • instance property activedTab was renamed to activeTabId; add computed property activeTab (4aea0b1)

# Features

  • restore: add option restore-watch to restore tab content on restore change (d040b23)

# BREAKING CHANGES

  • instance property activedTab was renamed to activeTabId
